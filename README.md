# 📟 مشروع كاشف الدخان الذكي (Smart Smoke Detector)

نظام أمان متكامل يعتمد على منصة **Arduino** لكشف الدخان والغازات الضارة وتفعيل إنذار فوري.

---

## 🏗️ المكونات المستخدمة (Hardware)
* **Arduino Board:** العقل المدبر للنظام.
* **MQ-2 Sensor:** مستشعر حساس للدخان والغاز.
* **Buzzer:** جرس إنذار صوتي للتنبيه.
* **LED Indicator:** لمبة بيان للحالة.

---

## ⚙️ كيف يعمل النظام؟
1. يقوم مستشعر **MQ-2** بقراءة جودة الهواء باستمرار.
2. إذا تخطت نسبة الدخان الحد المسموح به، يبدأ الإنذار الصوتي بالعمل.
3. يتم مراقبة البيانات مباشرة عبر الـ **Serial Monitor**.
